Aprenda a configurar a integração com e-Millennium


Credencias de acesso e-Millennium


Para realizar integração no e-Millennum há duas maneiras de autenticar

a) Basic Authentication requer usuário e senha do Windows
Usuário: “precisa ser Administrador”
Senha: ****
IP de internet Servidor Millenium
Liberação da Porta 6017, usada para integração

*) Se a empresa usar Dominio, deve criar um usuário no servidor Windows fora do domínio "usuário local" como Administrador, pois a autenticação por Basic Authentication pode ser negada por causa das regras de segurança do dominio.


b) Usuário do Millennium
Usando um usário do e-Millennium para authenticar, requer :
IP de internet Servidor Millennium, liberação da Porta 6017, usuário e senha criado no e-Millennium

*) Usar um usuário do e-Millennium é a maneira mais segura porque não precisa liberar acesso ao servidor Windows com plenos poderes para a empresa integradora.


* Observações: Se o servidor usar um proxi para acesso de internet, tem que ser feito um NAT, para redirecionar a porta 6017 para o IP da rede local do servidor e-Millennium.





Escopo de integração e cadastros


O escopo da integração Plugplayhub com e-Millennium é completa

* - Produtos + especificações e classificações do produto
* - Arvore de categorias
* - Preço de venda e estoques
* - Pedidos: Inclusão e atualização status dos pedidos
* - Documentos fiscais dos pedidos - xml e chave nfe
* - Enviar etiquetas para o e-Millennium: Mercado livre - Netshoes - B2W e etc.
* - Lista de vendedores e representantes
* - Tipos de Frete / Pedido / Condições de pagamento



Primeiro passo

a) Cadastre a vitrine que será integrada no e-Millennium, criar árvore de categorias, conforme será exposta na plataforma, se for uma vitrine de marketplace o ideal e separar de acordo com o departamento, masculino, feminino, infantil, criar sub-menus de acordoc com as caracterista do produto exemplo "tipo" Camiseta, calça e etc.



Configurar a integração dentro do PlugPlayhub?

b) Cadastre o webservice com as credeiciais de acesso ao e-Millennium, clique no icone da linx que é exibido no PlugplayHub, após o cadastro faça o teste de conexão, no botão "testar conexão". Se conectar com sucesso passe para a apróxima etapa.

c) Cadastre no hub qual será vitrine que será integrada, o hub irá listar todas as vitrines cadastradas no e-Millennium, não precisa se preocupar em pegar o ID com o suporte da Millennium.
